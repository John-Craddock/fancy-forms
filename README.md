# fancy-forms
SCSS library of fancy form elements

TO RUN
- npm run build-css
- npm run watch-css

Run the index.html file to see examples of each element.

FANCY FORM ELEMENTS
- input boxes
- checkboxes
- radio buttons
- select boxes
- radio buttons with images
- checkboxes with images
- fancy labels using radio buttons
- fancy labels using checkboxes
